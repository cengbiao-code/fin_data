# Issue Tracker

Issues and implementation tickets for this repo live in GitHub Issues:

```text
cengbiao-code/fin_data
```

Use the GitHub connector when available. If using the CLI, run commands from the repository root so the remote can identify the repo.

## Conventions

- Create implementation issues in dependency order.
- Use vertical slices that are independently verifiable.
- Reference blockers by GitHub issue number in the issue body.
- Prefer AFK-ready issues when the implementation can proceed without human decisions.
- Do not duplicate existing issues; search or inspect the open issue list first.

## Existing MVP Foundation Issues

Issues `#1` through `#11` covered the completed MVP foundation. Future work should create new issues for the real PDF extraction pipeline rather than reopening that plan.
